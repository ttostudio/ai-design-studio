**目的**: プロンプト入力 → 画像生成 → プレビュー確認の一連フロー

│  ③ プロンプト入力      │    （生成中: プログレス）      │

### ① テンプレートクイック選択（左ペイン上部）

| 要素 | 仕様 |

**受入基準対応**: AC-004（テンプレート選択でプリセット適用）

### ③ プロンプト入力エリア

| 要素 | 仕様 |
|------|------|
| メインテキストエリア | rows=5、placeholder=「画像の説明を入力（例: a futuristic city at night, neon lights）」|

| テンプレートプレフィックス | テンプレート選択時、テキストエリア上部に「プレフィックス: blog thumbnail, clean design,」バッジ表示 |

**制約**: FR-001（プロンプト入力必須）、NFR-004（1000文字以内）

### ⑤ プレビューエリア（右ペイン）

#### 状態別表示

**生成前（idle）**:
```
┌────────────────────────────────┐
│                                │
│         🎨                     │
│  プロンプトを入力して            │
│  画像を生成してください           │
│                                │
└────────────────────────────────┘
```
- 背景: `--color-bg-elevated`、ボーダー: ダッシュライン（`--color-border`）
- aspect-ratio: 選択サイズに合わせて動的変更

**生成中（queued / processing）**:
```
┌────────────────────────────────┐
│                                │
│  [██████████░░░░░░░░░░] 52%    │
│  生成中... ステップ 10/20       │
│                                │
└────────────────────────────────┘
```
- プログレスバー: height=8px、`--color-progress` 塗り、角丸 `--radius-sm`
- パーセンテージ: `--text-body-sm`、`--color-text-secondary`
- アニメーション: パルス（pulsing glow）

**完了（complete）**:
- 生成画像を `object-fit: contain` で表示
- 右上にアクションボタン群:
  - 「ダウンロード」（↓アイコン）
  - 「プロンプトをコピー」（コピーアイコン）
  - 「ギャラリーで見る」（外部リンクアイコン）

**エラー（error）**:
```
┌────────────────────────────────┐
│                                │
│  ⚠️  生成に失敗しました          │
│  {エラーメッセージ}              │
│                                │
│  [再試行]                       │
└────────────────────────────────┘
```

**受入基準対応**: AC-001（画像生成）、AC-002（進捗リアルタイム表示）、AC-006（エラー表示）

## 状態遷移図（生成フロー）

```
idle
  │ プロンプト入力 → 生成ボタンクリック
  ▼
queued                      ← POST /api/generate
  │ ジョブID 取得成功
  ▼
processing                  ← SSE /api/progress/:id
  │ progress: 0〜100%
  │ imageUrl イベント
  ├──────────────────────────► complete  → 画像表示
  │ status: 'error'
  └──────────────────────────► error     → エラー表示
                                            [再試行] → idle
```

ComfyUI オフライン時:
- `idle` 状態で生成ボタン disabled
- エラーバナー表示
- 状態バッジ: 赤（AC-006 対応）

## アクセシビリティ要件

| 要件 | 対応 |
|------|------|
| キーボード操作 | 全操作 Tab / Enter で完結可能 |
| スクリーンリーダー | `aria-label` / `aria-live` で生成状態を通知 |
| コントラスト比 | テキスト: 4.5:1 以上（WCAG AA 準拠） |
| フォーカス表示 | `outline: 2px solid --color-accent` |
| 画像 alt テキスト | プロンプト先頭100字を `alt` 属性に設定 |
