## スコープ

既存ギャラリー（`/gallery`）を強化し、生成画像のポートフォリオとして活用しやすくする。
認証機能は**対象外**（認証なし前提）。
ComfyUI連携・生成フロー自体は変更しない。

---

## 機能要件

### FR-01: ギャラリービュー強化

| ID | 要件 |
|----|------|
| FR-01-1 | グリッド表示とリスト表示を切替できる（現在はグリッドのみ） |
| FR-01-2 | グリッドの画像サイズを小/中/大の3段階で切替できる |
| FR-01-3 | 「もっと読み込む」ボタンによるページネーションは維持する（無限スクロールは本バージョンでは不採用） |

### FR-02: フィルタリング強化

| ID | 要件 |
|----|------|
| FR-02-1 | ワークフロー別フィルタ（flux-gguf / sd15）を追加 |
| FR-02-2 | テンプレート別フィルタは既存機能を維持・拡張（カスタムテンプレートも含む） |
| FR-02-3 | タグ付け機能を追加。DBに `tags TEXT[]` カラムを追加。詳細ページ/モーダルからタグを設定できる |
| FR-02-4 | ギャラリー画面でタグによるフィルタができる |
| FR-02-5 | 日付範囲フィルタは既存機能（今日/今週/今月）を維持 |
| FR-02-6 | ステータスフィルタを追加（success / error。デフォルトは success のみ表示） |

### FR-03: デザイン詳細ページ

| ID | 要件 |
|----|------|
| FR-03-1 | `/gallery/[id]` に個別ページを追加（モーダルとは別に専用URLでアクセス可能） |
| FR-03-2 | 表示内容: 画像、プロンプト、ネガティブプロンプト、ワークフロー、サイズ、ステップ数、CFGスケール、シード値、生成時間（ms）、生成日時 |
| FR-03-3 | 類似デザイン表示: 同一 `template_id` の直近6件を表示（template_idがない場合は非表示） |
| FR-03-4 | ダウンロードボタン（既存モーダルと同仕様） |
| FR-03-5 | 再生成ボタン: 同パラメータでトップページ（`/`）に遷移しフォームを自動入力 |
| FR-03-6 | ギャラリーカードから詳細ページへのリンクを追加（モーダルは引き続き表示、詳細ページボタンを追加） |

### FR-04: お気に入り機能

| ID | 要件 |
|----|------|
| FR-04-1 | 各ギャラリーカードにハートアイコンを表示し、クリックでお気に入りトグル |
| FR-04-2 | お気に入り状態は DB の `is_favorite BOOLEAN` カラムで管理（LocalStorageは使わない） |
| FR-04-3 | ギャラリーにお気に入りフィルタを追加（「お気に入りのみ表示」チェックボックス or トグル） |
| FR-04-4 | お気に入りトグルは即時反映（楽観的更新）。失敗時はロールバック |

## 受け入れ条件（AC）

### AC-01 ビュー切替
- [ ] グリッド/リスト切替ボタンがフィルターバーに表示される
- [ ] グリッド表示: `minmax(小=120px / 中=200px / 大=300px, 1fr)` でカードが並ぶ
- [ ] リスト表示: 1行に1件、左に画像サムネイル(64px)、右にプロンプト・日時が並ぶ
- [ ] 切替状態は `localStorage` に保存され、ページリロード後も維持される

### AC-02 フィルタ強化
- [ ] ワークフロードロップダウン（すべて/Flux-schnell/SD1.5）が表示・機能する
- [ ] タグフィールドから既存タグの選択またはフリーテキスト入力ができる
- [ ] ステータスフィルタ（success/error/すべて）が機能する
- [ ] API `GET /api/generations` に `workflow` / `tags` / `status` パラメータが追加される
- [ ] フィルタ変更時にリストが即時リセット（offset=0）される

### AC-03 詳細ページ
- [ ] `/gallery/[id]` にアクセスすると詳細ページが表示される（存在しないIDは404）
- [ ] FR-03-2 の全メタデータが表示される
- [ ] 類似デザインエリアに同一 `template_id` の画像が最大6件表示される
- [ ] ダウンロードボタン・再生成ボタンが動作する
- [ ] ギャラリーカードに「詳細ページを開く」ボタン（外部リンクアイコン）が表示される

### AC-04 お気に入り
- [ ] ギャラリーカード上にハートアイコンが表示される
- [ ] クリックするとDB更新（`PUT /api/generations/[id]/favorite`）が実行される
- [ ] お気に入り状態はハートアイコンの塗りつぶし有無で視覚的に区別される
- [ ] お気に入りフィルタON時、`is_favorite = true` の件のみ表示される
- [ ] 楽観的更新: UIは即時反映、API失敗時に元の状態に戻る

---

## ロールバック方針

- DBマイグレーション（`002_portfolio.sql`）は `ADD COLUMN IF NOT EXISTS` / `CREATE INDEX IF NOT EXISTS` のみ使用し、既存データを破壊しない
- マイグレーション失敗時は前バージョンのコードがそのまま動作する（カラム追加のみのため後方互換性あり）
- フロントエンド変更は機能フラグ不要（全新規コンポーネント or 既存コンポーネントへの追加）
- `/gallery/[id]` 追加は既存ルーティングに影響しない

---

## 対象外

- 認証・ログイン機能
- タグのサジェスト/オートコンプリート（初版では自由入力のみ）
- 無限スクロール（「もっと読み込む」ボタン方式を維持）
- コレクション機能（グループ化・フォルダ分け）
- 公開共有URL
